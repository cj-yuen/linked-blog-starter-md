>The encoding of $S$ constructed from binary tree $T$ is a prefix code
>	given alphabet $S$, for each letter $x\in S$ we follow the path from the root to the leaf labeled $x$ 
>		left $\rightarrow$ 0
>		right $\rightarrow$ 1
